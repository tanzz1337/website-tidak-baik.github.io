Ciri-Ciri Website Tidak Aman / Tidak Baik Dalam Pengelolaan nya

  • Tidak ada validasi input (rentan XSS, SQLi).
  • Menggunakan CMS dan plugin lawas.
  • Penggunaan username/password default.
  • Akses admin terbuka tanpa proteksi.
  • Tidak ada backup atau recovery.
  • Tidak menggunakan HTTPS.
  • Tampilan tidak responsif.
